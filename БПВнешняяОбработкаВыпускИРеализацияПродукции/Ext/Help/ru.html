Основные функции и приемы работы с обработкой

Обработка предназначена для типовой конфигурации Бухгалтерия предприятия 3.0 и основная ее функция - это создание документов:

  • Выпуск продукции (Отчет производства);
  • Реализация (акты, накладные, УПД).
Наполнение документов данными о товарах и контрагентах реализовано через сводный табличный документ (далее сводная таблица), для этого предлагается следующий порядок работы с обработкой.
  1. При открытии обработки необходимо выбрать дату создания документов и организацию от имени которой будут созданы документы.
  2. Заполнить списки контрагентов и номенклатуры в сводной таблице с помощью команды Заполнить.
  3. Проставить количество на пересечении строки с контрагентом и колонки с номенклатурой.
  4. Выполнить команды создания документов Выпуск продукции и Реализация через подменю Создать.

Отбор контрагентов и номенклатуры

Чтобы заполнить сводную таблицу только нужными позициями справочников контрагентов и номенклатуры, в обработку встроены гибкие отборы которые расположены на специальной панели настроек.
 Отборы можно устанавливать:

  • По иерархии справочников;
  • По подчиненным реквизитам справочников;
  • По реализациям - значения справочников выбираются из документов Реализаций за определенный период.
Данные отборы можно комбинировать между собой.

Список созданных документов

После выполнения любой команды создания документов открывается панель списка созданных документов. Перейти к списку созданных документов и вернуться к сводной таблице можно с помощью команды Документы.
 В списке документов доступны платформенные команды и команды печати наследуемые из конфигурации Бухгалтерия предприятия. Таким образом в списке документов можно выполнить групповые и одиночные действия проведения, отмены проведения, печати и прочее.
 Последний  созданный документ в списке выделяется светло зеленым цветом.

Заполнение реквизитов документов и дополнительные настройки

В данных обработки расположены настройки которые в явном виде переносятся в документы и влияют на их заполнение при создании.

Настройки командной панели.
  • Дата, Организация - используется для всех документов.
Настройки в панели настроек.
  • Номенклатурная группа - используется в документе Выпуск продукции, в случае, если у элемента номенклатуры не заполнен соответствующий реквизит.
  • Подразделение - используется для заполнения реквизита Подразделение в документе Выпуск продукции.
  • Счет затрат - используется для заполнения реквизита Счет затрат в документе Выпуск продукции.
  • Склад - используется для заполнения реквизита Склад в документах Выпуск продукции и Реализация.
  • Тип цены продажи - используется для заполнения цен в документе Реализация.
  • Тип цены плановой себестоимости - используется для заполнения плановой себестоимости в документе Выпуск продукции.
  • Заполнять материалы - будут заполнены материалы в документе Выпуск продукции.
  • Выписывать счета-фактуры - при создании документов Реализация, автоматически будут создаваться связанные счета-фактуры.
Дополнительные данные для заполнения реквизитов определяются в алгоритмах при создании документов. 
  • Подбор спецификации номенклатуры - выбирается основная, если такой нет, то не основная, но без пометки удаления.
  • Подбор договора контрагента - выбирается основной, если такого нет, то не основной, но без пометки удаления.
Для заполнения счетов учета и прочих сведений о номенклатуре в обработке используется типовой программный интерфейс конфигурации Бухгалтерия предприятия.
